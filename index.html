rr


Your shell script multi-tool





Install or Download (<3MiB static executable)

Go 1.16+

GOBIN=/usr/local/bin go install github.com/tongson/rr@0.15.0

Download for Linux amd64/x86_64

curl -L https://github.com/tongson/rr/releases/download/0.15.0/rr.linux-amd64 -o rr

Download for Linux 32-bit

curl -L https://github.com/tongson/rr/releases/download/0.15.0/rr.linux-i386 -o rr

Download for macOS Intel

curl -L https://github.com/tongson/rr/releases/download/0.15.0/rr.macos-intel -o rr

Download for macOS m1 (arm64)

curl -L https://github.com/tongson/rr/releases/download/0.15.0/rr.macos-m1 -o rr



shell script manager and runner inspired by rerun[1], bashing[2], and drist[3]
[1] github.com/rerun/rerun
[2] github.com/xsc/bashing
[3] bitreich.org/drist

Local (requires tar(1))
Local container (Linux only, requires nsenter(1) and tar(1))
Remote (requires OpenSSH, ssh(1) and tar(1))
